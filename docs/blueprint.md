# **App Name**: IDB Seguros Manager

## Core Features:

- User Authentication: Secure login and registration with admin role management.
- Sidebar Navigation: Intuitive sidebar for easy navigation through seguros, colectivos, contactos, correos, usuarios de portal, popups y archivos.
- Seguros Management: CRUD operations for seguros, including contacto, moneda, RIF, cobertura, and other relevant details.
- Colectivos Management: Manage colectivos associated with seguros, including names and descriptions.
- Contactos Management: Manage contactos related to seguros, with names, phone numbers, and comments.
- Correos Management: Manage correos associated with seguros, including names, email addresses, and comments.
- Archivos Management: Manage archivos associated with seguros, including file uploads and storage URLs.

## Style Guidelines:

- Primary color: #003b5c (a dark blue) for headers and footers to provide a strong contrast.
- Secondary color: #0077bc (a vibrant blue) for main elements to convey trust and stability.
- Background color: #ffffff (white) to provide a clean and calm backdrop.
- Font: 'Inter', a sans-serif typeface for both headlines and body text, chosen for its modern and readable appearance.
- Use a consistent set of icons to represent different seguro-related categories.
- A clean, well-spaced layout for displaying seguro information.
- Subtle animations to indicate loading or successful saving of seguro details.
